## 0.1.0

- Restructured the project as a publishable Flutter package.
- Added public API: `LiquidGlassSwitch`, `LiquidGlassSwitchFace`, and `LiquidGlassSwitchMotion`.
- Added controlled interaction model with spring animation and drag flick threshold.
- Added accessibility semantics for switch state and activation.
- Added `example/` app that demonstrates the package.
- Updated documentation and package metadata for pub.dev.
